Featured Projects

Project Archives

Frog Animation

Modeling & Rigging

For this project I began with creating the base mesh of a frog using Autodesk Maya. Following this I was able to bring this mesh over into Mudbox to sculpt the mesh making it take on more life-like qualities. Following this I brought the mesh back to Maya to create the rig. Upon finishing the rig of the frog I animated a simple jump. This video contains the actual model that a user would want as well as the control curves and the rig itself.
