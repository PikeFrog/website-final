Frog Animation

Modeling & Rigging

For this project I began with creating the base mesh of a frog using Autodesk Maya. Following this I was able to bring this mesh over into Mudbox to sculpt the mesh making it take on more life-like qualities. Following this I brought the mesh back to Maya to create the rig. Upon finishing the rig of the frog I animated a simple jump. This video contains the actual model that a user would want as well as the control curves and the rig itself.

Avian Character Rig

This character was created starting from a cube. The body was then proportioned out in Maya before being brought into Mudbox to begin the sculpting process. Following the sculpting of the anatomy of the body I began to sculpt in the armor pieces of the model. After the armor was sculpted onto the mesh I began the texturing process in Mudbox. The last process was the rigging which took place in Maya along with a simple walk cycle to test the rig.

DEC Hudson River Project

For this project I worked as a part of a team of artists, students and other people who each had different responsibilities. My responsibilities were to create game object assets in Autodesk Maya and import them into Unity. Following this I created scripts which correlated with the intended game objects so they worked properly. More specifically I created the environment of the scene as well as diatom models which flowed in a physics based system. I also created fish models which took data from .csv files and played different animations and reacted accordingly to the data that was read.
